R E S U M E N

El Proyecto principalmente fue creado para solucionar distintas problematicas que enfrentan los campesinos en su vida laboral y sustento para su vida diaria, basicamente busca y buscamos ayudar a los campesinos ubicados en una localidad cercana a bogota, que enfrentan problematicas laborales, sus productos y cosechas que estan listos para la comercializacion, sufren una depreciacion, esto es debido a que en el siguiente proceso de logistica se encuentran los intermediarios, los cuales ofrecen un muy bajo precio a sus cosechas y productos para su conveniencia, de manera en la que el campesino se ve afectado directamente, y por lo cual su economia Tambien. El programa omite este proceso conectando al campesino con los distribuidores y compradores directamente, dandole el control de sus precios hacia el Mercado

T E C N O L O G I A S

HTML, CSS, SQL

En resumen, HTML se encarga de definir la estructura y el contenido de la página web, CSS se utilizamos para aplicar estilos y diseño visual a los elementos HTML, y MySQL se utilizamos para almacenar y gestionar datos en la base de datos y hacer las tablas de Entidad relación del proyecto. Estas tecnologías son fundamentales para el desarrollo web y trabajan juntas para crear aplicaciones web completas y funcionales.
